# 마크다운 예제

이 문서는 마크다운 기반 콘텐츠 관리 시스템의 예제 문서입니다.

## 기능 소개

1. 프론트매터 지원
2. 코드 하이라이팅
3. GFM(GitHub Flavored Markdown) 지원

### 코드 예제

```javascript
function hello() {
  console.log("Hello, World!");
}
```
